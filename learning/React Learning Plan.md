
# **Fast-Track 3-Month React Learning Plan (Week-Wise)**

**Total Duration:** 12 weeks (~3 months)
**Daily Time Commitment:** 2–3 hours (more on weekends)
**Goal:** From zero to job-ready React developer with portfolio projects

---

## **Week 1: Pre-Requisites (HTML, CSS, JS Basics)**

**Objectives:**

* Learn HTML tags, semantic elements, forms, links, images
* Learn CSS basics: selectors, colors, fonts, box model, flexbox, grid
* JavaScript basics: variables, types, operators, loops, functions, arrays, objects
* DOM manipulation: select elements, change content, handle events

**Projects/Exercises:**

* Build a personal webpage (name, hobbies, gallery of his drawings)
* Small interactive JS exercise: button click changes color/text

**YouTube Channels:**

* FreeCodeCamp (HTML, CSS, JS full course)
* Web Dev Simplified (JS DOM tutorials)

---

## **Week 2–3: React Basics**

**Objectives:**

* React setup (Create React App / Vite)
* JSX syntax, embedding JS in JSX
* Components, props, state
* Event handling, conditional rendering
* Lists rendering, keys
* Simple forms

**Projects/Exercises:**

* Mini **Drawing Gallery** (show images of his drawings using components)
* Simple to-do list

**YouTube Channels:**

* Codevolution (React Basics & Components)
* JavaScript Mastery (Beginner React Projects)

---

## **Week 4–5: Intermediate React**

**Objectives:**

* React Router for multiple pages
* Fetch data from APIs (Python backend or dummy JSON API)
* Lifting state up and props drilling
* Component reusability
* Introduction to functional programming patterns in JS (map, filter)

**Projects/Exercises:**

* Portfolio Website with multiple pages
* Recipe App fetching data from API and displaying cards

**YouTube Channels:**

* Codevolution (React Router, API integration)
* JavaScript Mastery (Projects and API usage)

---

## **Week 6–7: Advanced React Concepts**

**Objectives:**

* React Hooks: useState, useEffect, useRef, custom hooks
* Context API for state management
* Forms with validation
* Error boundaries, conditional rendering improvements
* Introduction to optimization (memo, lazy loading)

**Projects/Exercises:**

* Interactive Drawing App: allow drawing, saving, sharing images
* Quiz or small game app using React state and hooks

**YouTube Channels:**

* Codevolution (Advanced Hooks & Context)
* Dev Ed (React hooks & animations)

---

## **Week 8: React UI Libraries & Styling**

**Objectives:**

* Tailwind CSS / Material UI basics
* Component customization
* Basic animations, hover effects, transitions

**Projects/Exercises:**

* Upgrade previous apps using Tailwind CSS or Material UI
* Focus on visual polish, layout, typography, and color

**YouTube Channels:**

* The Net Ninja (Tailwind + React tutorials)
* Dev Ed (UI/UX & animations)

---

## **Week 9–10: Full Projects Development**

**Objectives:**

* Combine all learned skills
* Build **2–3 polished projects**

  * Portfolio Website
  * Interactive Drawing App
  * Small productivity or game app

**Tips:**

* Design first (sketch UI) → Code second
* Push projects to **GitHub**

**YouTube Channels:**

* JavaScript Mastery (Full React Projects)
* Dev Ed (Creative UI project tutorials)

---

## **Week 11: Deployment & Portfolio Completion**

**Objectives:**

* Deploy apps using Netlify or Vercel
* Create portfolio site to showcase all projects
* Add responsive design (mobile friendly)

**Projects/Exercises:**

* Deploy Portfolio Website + Projects
* Prepare project README for GitHub

**YouTube Channels:**

* Net Ninja (React Deployment Tutorials)
* JavaScript Mastery (Portfolio Deployment)

---

## **Week 12: Job Preparation & Review**

**Objectives:**

* Review all concepts (JS, React, Hooks, APIs, Routing)
* Polish portfolio UI/UX
* Prepare common interview questions & coding challenges

**Projects/Exercises:**

* Final touches on portfolio
* Practice coding questions on LeetCode / HackerRank (JS basics)

**YouTube Channels:**

* CS Dojo (Interview prep basics)
* Codevolution (React interview questions)

---

### **Creativity Growth Tips Throughout**

* Always **sketch UI ideas before coding**
* Turn his drawings into app components
* Experiment with **colors, typography, animations**
* Try mini “art challenges” weekly to keep creative muscles strong
* Study UI/UX inspirations on **Dribbble / Behance**

---

| Week | Focus              | Goal                                     |
